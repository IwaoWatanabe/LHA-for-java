このパッケージは java.lang.reflect パッケージによるリフレクションを支援するクラスを提供する。 主に jp.gr.java_conf.dangan.lang.reflect.Type による文字列からプリミティブ型へや 相互変換可能なプリミティブ型(byte,short,int,long,float,double)の変換、と 変換を利用した型検査を使用して jp.gr.java_conf.dangan.lang.reflect.Factory と jp.gr.java_conf.dangan.lang.reflect.MethodUtil で比較的少ないステップ数で インスタンスの生成やメソッドの起動を支援する。
